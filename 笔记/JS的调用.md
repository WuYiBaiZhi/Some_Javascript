行内式

a标签

```javascript
<a href="javascript: alert('Hello World')">点我试试</a>
```

alert 在浏览器弹出提示框

非a标签

```javascript
<div onclick="alert('Hello Words')">点我一下</div>
```

onclick 为点击属性

内嵌式

```javascript
        <script>
            
        </script>
```

外联式

```javascript
<script type="text/javascript" src="js文件地址">
            
        </script>
```
